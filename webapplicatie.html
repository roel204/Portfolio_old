(NL) Senioren Zorg Webapplicatie.

Ik heb samen met mijn team tijdens CLE-3 een web-applicatie gemaakt. Het doel van onze website is om de doelgroep: Senioren meer informatie te geven over de zorg.
Op onze pagina staat een vrouw genaamd Hennie, zij geeft de gebruiker meer info over de pagina door tegen de gebruiker te praten.

Home pagina:

Dit is de begin pagina van onze web-applicatie.
Op het prikbord staan 4 knoppen die naar elke pagina van het team gaan.

Mijn pagina:

Dit is de pagina die ik heb gemaakt, hier staat info over de rechten in de zorg en nog andere dingen over rechten en wetgeving.
Gebruikers kunnen hun favorite artikelen een ster geven om te markeren dat ze favoriet zijn.

Informatie pagina:

Alle info staat aangegeven in een dialog die geopend word als er op het kaartje geklikt word, het kan gesloten worden door op het kruisje te drukken.
Er staat een korte beschrijving van het onderwerp en een link naar de bron van de informatie.

De Code:

De website maakt gebruik van HTML, Javascript en een beetje PHP om het te laten werken. Ook gebruikt de website een vorm van JSON en Localstorage om gegevens in op te slaan.
De code van de website is te zien op mijn GitHub pagina.
